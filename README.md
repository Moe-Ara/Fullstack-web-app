<!-- GETTING STARTED -->
## Getting Started

To Try running the app on your local machine please do the following

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```
* flask
  ```sh
  pip install flask
  ```
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Moe-Ara/Fullstack-web-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the App
   ```sh
   yarn start
   yarn start-api
   ```
